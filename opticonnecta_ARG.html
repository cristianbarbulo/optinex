Resolviendo tus Dudas

¿Esto es solo un chatbot con respuestas predefinidas?

Definitivamente no. OptiConnect no sigue un script. Es un sistema de múltiples agentes de IA que razonan, recuerdan conversaciones, se integran con tus sistemas de datos y aprenden para volverse más expertos con cada interacción.

Mi empresa no usa HubSpot, ¿aún así puedo usarlo?

¡Por supuesto! Nuestra plataforma está construida con una arquitectura de adaptadores. Podemos integrarnos con casi cualquier CRM o sistema que tenga una API disponible, ya sea Salesforce, Zoho, o incluso un sistema desarrollado a medida.

¿Puede manejar imágenes o solo texto?

Sí, puede manejar imágenes. Un cliente puede enviar una foto de un producto que vio en la vidriera, y el agente de IA la analizará para identificarlo, consultar el stock en tu sistema y dar el precio.

¿Qué tan seguro es el manejo de los datos?

La seguridad es fundamental. Cada cliente de OptiConnect opera en una infraestructura de servidor y base de datos completamente aislada y dedicada. No hay recursos compartidos ni posibilidad de cruce de datos.

¿Cuál es la diferencia real entre los planes?

La diferencia principal es la capacidad para realizar acciones complejas y proactivas. Cada plan desbloquea nuevos "módulos" y agentes de IA con capacidades más avanzadas, desde el agendamiento y la conexión a catálogos, hasta el manejo de voz y pagos.

¿Qué implica el proceso de implementación?

Es un proceso guiado y sencillo. Primero, tenemos una reunión de descubrimiento para entender tu negocio. Luego, nos das acceso a la información clave (catálogo, preguntas frecuentes). Nuestro equipo configura y entrena a los agentes de IA y finalmente, vinculamos el sistema a tu número de WhatsApp. Normalmente, puedes estar operativo en pocos días.

¿Qué pasa si el bot no sabe responder?

El sistema está diseñado para la colaboración. Si la IA detecta una consulta que no puede manejar con la información que tiene, o si el cliente solicita explícitamente hablar con una persona, la conversación se pausa y se notifica a un miembro de tu equipo para que pueda tomar el control desde su propio WhatsApp o panel de control.

¿Qué sucede si excedo el límite de conversaciones de mi plan?

Tu servicio nunca se detendrá. Si excedes el límite, te contactaremos para ofrecerte dos opciones: pasar al siguiente plan para obtener un mejor costo por volumen o pagar un pequeño excedente por cada conversación adicional. Siempre tendrás el control de tus costos.

¿Entonces esto es un CRM o un Chatbot?

Es ambas cosas y mucho más. Es un Centro de Comando Inteligente. La IA (el chatbot) se encarga de la primera línea 24/7, y la plataforma (el CRM de conversaciones) le da a tu equipo humano las herramientas para colaborar, supervisar y gestionar todas las interacciones desde un único lugar.
